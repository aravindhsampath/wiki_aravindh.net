# Freebsd

FreeBSD related sysadmin stuff

## Updating FreeBSD

```bash
pkg update
pkg upgrade
```

## Linux style free command in FreeBSD

```bash
root@ironbank01:~ # pkg install freecolor
root@ironbank01:~ # freecolor -mo
             total       used       free     shared    buffers     cached
Mem:        383051     373927       9123          0          0          0
Swap:         4096          0       4096
```
